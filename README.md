# Coffee-Shop-Management-System
A coffee shop management system in Java. This project contains two interfaces: Client and Admin. This project demonstrates the OOP and Advanced Java concepts. The client can order a coffee and get a bill. The admin can add, delete, modify rates and calculate his turnover.
